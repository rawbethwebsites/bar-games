## Variant: Minimal Ritual

### Design stance
Dark theatre, huge type, calm hierarchy. The game feels like a curated ritual rather than a noisy app.

### Key choices
- **Layout:** Centered, single-column, oversized headline.
- **Typography:** Cormorant Garamond for display, Inter for UI.
- **Color:** Near-black background, warm off-white text, single gold accent.
- **Interaction:** Pill buttons with subtle border glow on hover.

### Trade-offs
- Strong at: readable from across a room, feels intentional and premium.
- Weak at: less immediate game energy; relies on animation/sound to feel alive.

### Best for
A refined dinner where the screen should look beautiful even when nobody is playing.
