## Variant: Classy Courtroom

### Design stance
Formal, celebratory courtroom — gold seal, ornate frame, serif display type. Feels like a toast to the new barrister.

### Key choices
- **Layout:** Centered card inside a glowing gold frame.
- **Typography:** Cinzel for headings (courtroom serif), Inter for body.
- **Color:** Mahogany background, parchment text, brass/gold accents, prosecution red and defence blue reserved for gameplay.
- **Interaction:** Hover lift on mode cards.

### Trade-offs
- Strong at: premium event feel, on-theme for Call to Bar dinner.
- Weak at: could feel too formal for a party game if overused elsewhere.

### Best for
A classy dinner setting where the game is part of a ceremonial evening.
